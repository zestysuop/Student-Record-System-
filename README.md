# 🎓 Student Record System

A simple and responsive Student Record System built using HTML, CSS, and JavaScript. The application allows users to add and delete student records dynamically without refreshing the page.

---

## 🚀 Features

- Add new student records
- Store Name, Roll Number, Course, and Marks
- Display records in a table
- Delete student records
- Responsive and clean UI
- Beginner-friendly JavaScript project

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
